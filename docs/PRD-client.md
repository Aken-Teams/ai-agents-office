# AI Agents Office — 產品需求文件（客戶版）

> 版本：1.0｜日期：2026-03-25

---

## 1. 產品簡介

**AI Agents Office** 是一套企業級的 AI 智慧文件生成平台。您只需要用自然語言描述需求，系統就會自動幫您產出 PowerPoint 簡報、Word 文件、Excel 報表、PDF 文件等商業文件。

### 為什麼選擇 AI Agents Office？

- **資料不離開公司** — 所有運算與檔案儲存都在您的本機環境，不經過第三方雲端
- **對話即文件** — 像跟助理講話一樣描述需求，系統自動完成排版與內容
- **一句話多份文件** — 複雜需求自動拆解，同時產出簡報、報告、數據表
- **即時看到過程** — 全程透明，您可以看到 AI 正在做什麼、用了什麼工具
- **改到滿意為止** — 每次修改自動保留舊版，隨時回溯之前的版本

## 4. 核心功能

### 4.1 對話式文件生成

透過自然語言對話，描述您需要什麼文件，系統會自動完成：

> **您：**「幫我做一份 Q1 業績報告的簡報，包含營收趨勢圖、各部門貢獻占比、和下季度目標」
>
> **AI：** 自動研究 → 規劃大綱 → 產出 PowerPoint 簡報

### 4.2 智慧多步驟處理

面對複雜需求，系統會自動拆解為多個步驟：

1. **研究** — 上網蒐集最新資料（如果需要的話）
2. **規劃** — 產出文件結構與大綱
3. **生成** — 自動排版並產出成品
4. **審查** — 檢查內容品質與一致性

整個過程您都可以即時看到每個步驟的進度。

### 4.3 上傳資料輔助

您可以上傳自己的檔案（CSV、Excel、圖片等），AI 會根據您的資料來生成文件：

- 上傳銷售數據 → 自動生成趨勢圖表的簡報
- 上傳會議紀錄 → 整理成結構化的報告
- 上傳多份文件 → 進行跨文件分析與摘要

系統會自動掃描上傳檔案的安全性。

### 4.4 版本管理

每次修改都會自動保留前一版：

- 不滿意？隨時下載之前的版本
- 多次迭代？所有版本都完整保留
- 版本歷史清楚可追溯

### 4.5 即時進度透明

生成過程中，您可以看到：

- AI 目前正在做什麼（搜尋資料、寫入檔案、分析數據...）
- 哪些 Agent 被派出任務
- 每個步驟的完成狀態
- Token 使用量即時更新

### 4.6 檔案預覽

產出的文件可以直接在瀏覽器中預覽，不需要另外安裝軟體：

- PDF 直接預覽
- HTML 簡報即時播放
- 所有格式一鍵下載

## 5. 安全性

### 5.1 資料保護

| 保障項目 | 說明 |
|----------|------|
| 本機運算 | 所有 AI 運算都在您的環境中執行，資料不上傳至第三方 |
| 使用者隔離 | 每位使用者的檔案完全隔離，互相無法存取 |
| 對話隔離 | 每次對話的工作空間獨立，避免資料交叉 |
| 輸入過濾 | 自動偵測並攔截惡意指令，保護系統安全 |
| 上傳掃描 | 所有上傳檔案自動進行安全掃描 |
| 審計日誌 | 完整記錄所有安全事件，可供追溯 |

### 5.2 權限控管

- 使用者帳號 + 密碼認證
- 支援 Google 企業帳號單一登入 (SSO)
- 管理員獨立後台，可控制使用者權限
- 登入失敗鎖定保護（5 次失敗 → 15 分鐘冷卻）

## 8. 部署需求

### 8.1 硬體需求

| 項目 | 最低需求 | 建議配置 |
|------|----------|----------|
| CPU | 4 核心 | 8 核心以上 |
| 記憶體 | 8 GB | 16 GB 以上 |
| 硬碟 | 20 GB | 100 GB SSD |
| 網路 | 有網路連線（AI 引擎需要） | 穩定的寬頻網路 |

### 8.2 軟體需求

- 作業系統：Windows 10+、macOS、Linux
- 瀏覽器：Chrome、Edge、Firefox、Safari（最新版本）

### 8.3 安裝時間

首次安裝約 30 分鐘，含環境設定與初始化。
